JSONPath Evaluator

Query JSON data using JSONPath expressions. Supports $, ., [], [*], slices, recursive descent, filters, and wildcards.

Examples:
$ (root) $.store.book[*].title $.store.book[0] $.store.book[-1] $.store.book[0:2] $..author (recursive) $.store.* (wildcard) filter: price < 9 $..price filter: author == Orwell
Results
JSONPath Syntax Reference
$ — Root element
. — Child operator
.. — Recursive descent
* — Wildcard (all children)
[n] — Array index
[-1] — Last element
[0:2] — Array slice
[*] — All array elements
['key'] — Bracket notation
?(@.x > 1) — Filter expression
@ — Current node (in filter)
