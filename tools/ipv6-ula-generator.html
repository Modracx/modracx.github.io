About This Address

RFC 4193 Unique Local Addresses use the FC00::/7 prefix. The L bit is set to 1 (FD prefix), indicating a locally assigned address. The 40-bit global ID is randomly generated to minimize collision probability when connecting networks. These addresses are not routable on the global internet.
